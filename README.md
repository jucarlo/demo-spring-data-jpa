# demo-spring-data-jpa

Simple Spring Boot app that return a greeting

## Contains concepts as
+ JPA Entity
+ JPA Repository
+ DTO
+ Mockito Mocks
+ Spring MVC
+ Annotations: @RestController
+ Spring Mock-MVC

## Tools
+ Postman
+ GET http://localhost:8080/api/v1/greeting
